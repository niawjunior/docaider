CSS Isolation Test Page

This page has intentionally conflicting CSS styles to test if the Docaider embed widget is properly isolated and doesn't affect (or get affected by) the host website's styles.

Host Website Styles:

  • Font: Comic Sans MS
  • Primary Color: Red (#ff0000)
  • All divs have red borders
  • All elements use content-box sizing
  • Large line-height and font-sizes

Expected Behavior:

  • ✅ Host website styles remain unchanged
  • ✅ Embed widget uses Prompt font (not Comic Sans)
  • ✅ Embed widget uses its own colors (not red)
  • ✅ Embed widget has no red borders
  • ✅ Embed widget uses border-box sizing

Instructions:

  1. Open the browser console to check for errors
  2. Click the chat widget button in the bottom-right corner
  3. Verify the widget opens with correct styling
  4. Verify this page's styles are NOT affected
  5. Type a message and send it
  6. Check that fonts, colors, and spacing are correct in both the widget and the page
