Docaider Embedded Chat Test

About This Test Page

This page demonstrates the Docaider embedded chatbox functionality. The chatbox should appear in the bottom-right corner (or as configured in the embed settings).

If you're seeing this page but not the chatbox:

  • Make sure embedding is enabled for the knowledge base
  • Check that the knowledge base ID in the script tag is correct
  • Verify that the embed.js file is properly served from the public directory

Test Content

This is some sample content to test how the chatbox integrates with a regular webpage. The chatbox should float above this content without interfering with the page layout.

You can ask the AI questions about the knowledge base that's connected to this embedded chat.

Testing Instructions

Click the "Run Embedded Chatbox Test" button in the top-right corner to run automated tests on the embedded chatbox.

Test results will appear in your browser's console (press F12 to open developer tools).
