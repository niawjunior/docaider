Docaider Embed Test Page

This page demonstrates the simplified embed script integration with a single script tag.

How It Works

The chat widget is loaded with a single script tag that includes all necessary configuration as data attributes:

<script 
  src="./embed.js"
  data-kb-id="c962805f-2d89-4e3f-9334-e4f7a1d5cb30"
  data-title="Docaider Test"
  data-primary-color="#7C3AED"
  data-position="bottom-right"
  data-width="350px"
  data-height="500px"
  data-welcome-message="Hello! This is a test of the simplified embed script."
  data-placeholder="Ask me anything..."
  data-button-text="Chat with Docaider"
  data-show-button-text="true"
  data-text-color="#FFFFFF">
</script>

Try It Out

You should see a chat button in the bottom right corner of this page. Click it to open the chat widget.
